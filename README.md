# README

## Examples

### Dependencies

- Arch: `sudo pacman -S libx11 libxinerama`
- Void: `sudo xbps-install libX11-devel libxinerama-devel`
- Gentoo: `x11-libs/libX11 x11-libs/libXinerama`
- Ubuntu/Debian: `sudo apt-get install libx11-dev libxinerama-dev`
- Fedora: `sudo dnf install libX11-devel libXinerama-devel`
- Alpine: `sudo apk add libx11-dev libxinerama-dev`
- openSUSE: `sudo zypper install libX11-devel libXinerama-devel`
- Manjaro: `sudo pacman -S libx11 libxinerama`
